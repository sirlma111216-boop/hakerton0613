# Hakerton 0613

이 프로젝트는 간단한 웹 애플리케이션입니다. HTML, CSS, JavaScript를 사용하여 구성되어 있습니다.

## 실행 방법
따로 서버를 구축할 필요 없이, 브라우저에서 `index.html` 파일을 열면 바로 실행해 볼 수 있습니다.

## 배포 가이드 (GitHub Pages)

이 프로젝트는 정적 파일(HTML, CSS, JS)로만 구성되어 있으므로 **GitHub Pages**를 사용해 매우 쉽게 무료로 배포할 수 있습니다.

### 배포 방법:
1. GitHub 저장소 [hakerton0613](https://github.com/sirlma111216-boop/hakerton0613) 페이지에 접속합니다.
2. 상단 메뉴에서 **Settings (설정)** 탭을 클릭합니다.
3. 좌측 사이드바에서 **Pages**를 클릭합니다.
4. **Build and deployment** 섹션의 Source 옵션이 `Deploy from a branch`로 되어 있는지 확인합니다.
5. **Branch** 설정 부분에서 `None`으로 되어 있는 드롭다운을 클릭해 `main` (또는 `master`) 브랜치를 선택합니다. 폴더 옵션은 `/ (root)` 상태로 둡니다.
6. **Save (저장)** 버튼을 누릅니다.
7. 몇 분 정도 기다린 후, 상단에 표시되는 URL (예: `https://sirlma111216-boop.github.io/hakerton0613/`)에 접속하면 웹사이트가 배포된 것을 확인할 수 있습니다!
